SqLite

concepto

SQLite es un sistema de gestión de bases de datos relacional (RDBMS) muy ligero y autónomo, que se distingue porque no funciona como un servidor independiente, sino que se integra directamente en las aplicaciones. Es uno de los motores de bases de datos más usados en dispositivos móviles, navegadores web y aplicaciones de escritorio.

caracteristicas generales

Código abierto y gratuito (licencia pública de dominio). Multiplataforma: funciona en Windows, Linux, macOS, Android, iOS, etc. Base de datos embebida: no requiere instalación de servidor, se integra directamente en el programa. Ligera: la biblioteca de SQLite ocupa menos de 1 MB. Usa un solo archivo (.sqlite o .db) para guardar toda la base de datos (tablas, índices, registros, etc.).
