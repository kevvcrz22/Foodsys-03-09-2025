Gestor de bases de datos

MongoDB

MongoDB es una base de datos NoSQL de código abierto, orientada a documentos. En lugar de almacenar los datos en tablas como en las bases de datos relacionales, MongoDB guarda los datos en documentos BSON (una extensión de JSON), que son estructuras flexibles que pueden contener distintos tipos y estructuras de datos.
