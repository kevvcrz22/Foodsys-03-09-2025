Gestor de bases de datos

SQL Server

SQL Server es un sistema de gestión de bases de datos relacional (RDBMS) desarrollado por Microsoft. Se utiliza comúnmente en entornos empresariales, especialmente aquellos basados en Windows. Su lenguaje de consulta principal es T-SQL (Transact-SQL), que extiende el SQL estándar con funcionalidades adicionales. SQL Server tiene una excelente integración con otras herramientas de Microsoft como .NET, Power BI y Azure. Ofrece versiones gratuitas como SQL Server Express, y versiones comerciales más avanzadas.
