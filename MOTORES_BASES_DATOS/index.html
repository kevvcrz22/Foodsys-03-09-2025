Gestor de bases de datos

¿QUE ES?

Es un software que actúa como interfaz entre el usuario y la base de datos, permitiendo crear, administrar y gestionar la información, también asegura que los datos sean accesibles de manera eficiente, segura e íntegra.

¿CUAL ES SU FUNCIÓN?

Algunas de sus funciones principales son: las creaciones de las bases de datos, gestión de usuarios, control de concurrencia y mantenimiento.
