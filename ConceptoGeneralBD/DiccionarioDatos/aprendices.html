Diccionario de Datos de FOODSYS

Nombre APRENDICES
Creación 30/07/2025
Descripción Registro de Aprendices de la empresa FOODSYS
Campo Tipo de dato Tamaño Descripción
ID_aprendiz INT 10 Se registra un numero unico e irrepetible
Tipo de Documento VARCHAR 2 Se registra el tipo de documento
Documento INT 20 Se registra el numero de documento
Primer Nombre VARCHAR 20 Se registra el primer nombre del aprendiz
Segundo Nombre VARCHAR 20 Se registra el segundo nombre del aprendiz
Primer Apellido VARCHAR 20 Se registra el primer apellido del aprendiz
Segundo Apellido VARCHAR 20 Se registra el segundo apellido del aprendiz
Genero VARCHAR 15 Se registra el tipo de genero del aprendiz
Correo eletronico VARCHAR 50 Se registra el correo del aprendiz
Telefono de Contacto INT 20 Se registra el numero de contacto
Interno VARCHAR 5 Se registra si pertenece o no al centro de Convivencia
Estado VARCHAR 15 Se registra si el estado de formacion en la que se encuentra (Activo, condicionado, Inactivo)
Excepciones VARCHAR 5 Se registra si tiene o no preferencias
