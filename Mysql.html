Gestor de bases de datos

Mysql

MySQL es un sistema de gestión de bases de datos relacional (RDBMS) de código abierto muy popular, utilizado para almacenar, organizar y acceder a datos de manera eficiente, especialmente en aplicaciones web. Se basa en el lenguaje SQL (Structured Query Language) y permite realizar operaciones básicas conocidas como CRUD: para crear datos se usa INSERT INTO usuarios (id, nombre) VALUES (1, 'María');, para consultar se emplea SELECT * FROM usuarios WHERE id = 1;, para actualizar se utiliza UPDATE usuarios SET nombre = 'María López' WHERE id = 1;, y para eliminar datos se ejecuta DELETE FROM usuarios WHERE id = 1;. MySQL destaca por su rapidez, fiabilidad y compatibilidad con múltiples plataformas, siendo una opción clave en entornos LAMP (Linux, Apache, MySQL, PHP).
