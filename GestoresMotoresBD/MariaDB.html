mariaDB

Concepto

MariaDB es un sistema de gestión de bases de datos relacional (RDBMS) creado como bifurcación (fork) de MySQL, desarrollado por los mismos fundadores originales de este. Se diseñó para mantener compatibilidad con MySQL, pero añadiendo mejoras de rendimiento, seguridad y nuevas funcionalidades.

caracteristicas generales

Software libre y de código abierto (licencia GPL v2). Compatibilidad con MySQL en comandos, librerías y conectores. Multiplataforma: funciona en Windows, Linux, macOS y otros sistemas Unix. Alta escalabilidad: soporta desde pequeñas aplicaciones hasta grandes sistemas empresariales. Comunidad activa y soporte empresarial a través de MariaDB Corporation.
