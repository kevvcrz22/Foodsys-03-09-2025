Gestor de bases de datos

PostgreSQL

PostgreSQL es un sistema de base de datos relacional de código abierto, altamente extensible y compatible con el estándar SQL. Es muy popular en entornos que requieren integridad de datos, consultas complejas y personalización avanzada. A diferencia de SQL Server, es completamente gratuito y puede correr en múltiples sistemas operativos, incluyendo Linux, Windows y macOS.
